# 图文排版工具

一个开源、离线运行的中文长文排版工具。它可以把文章自动分页为适合社交媒体发布的竖版图片，并支持标题、作者信息、富文本标记、插图和批量导出 PNG。

项目不依赖数据库或外部 API。文章内容和草稿只保存在你的浏览器中，不会上传到远程服务器。

## 功能

- 将长文章自动分页为竖版图片
- 默认输出 1080 × 1440 画布，并支持自定义尺寸
- 设置标题、作者、日期、头像和页脚
- 使用 `Ctrl+B` 加粗选中文字
- 为当前行或选中多行添加圆点列表
- 使用 Markdown 标题和三反引号代码块
- 保留手动换行，并支持手动插入分页
- 插入、删除图片并保持原始宽高比
- 调整字体、字号、间距、边距和配色
- 将所有页面批量导出为 PNG
- 自动将正文草稿保存到浏览器本地存储

## 运行要求

- [Node.js](https://nodejs.org/) 18 或更高版本
- 推荐使用最新版 Chrome 或 Edge

项目没有第三方 npm 依赖，下载后可以直接启动。

## 快速开始

进入项目目录后运行：

```bash
npm start
```

在浏览器中打开：

```text
http://localhost:4174
```

如果端口已被占用，可以指定其他端口：

```bash
PORT=8080 npm start
```

Windows PowerShell：

```powershell
$env:PORT=8080
npm start
```

**成功标志：** 终端显示本地访问地址，浏览器打开后可以看到左侧编辑区和右侧分页预览。

## 使用方法

1. 在左侧正文框输入或粘贴文章。
2. 设置标题、作者、画布、字体、间距和颜色。
3. 使用正文上方按钮添加加粗、列表、代码、图片或分页。
4. 在右侧实时检查每一页的排版效果。
5. 点击导出按钮，将全部页面保存为 PNG。

支持的正文标记：

````markdown
# 一级标题

这是包含 **加粗文字** 的正文。

```js
console.log('代码模块');
```
````

单独一行输入 3 个空格，可以从下一页继续排版。

## 浏览器兼容

Chrome 和 Edge 支持选择文件夹后批量保存图片。其他浏览器如果不支持文件夹选择接口，工具会改为逐张下载。

## 隐私说明

- 不需要注册账号。
- 不连接外部 API。
- 不会上传文章、头像或插图。
- 正文草稿保存在浏览器的 `localStorage` 中。
- 用户选择的本地图片只在当前浏览器页面中处理。

清除浏览器网站数据会同时删除本地保存的正文草稿。
